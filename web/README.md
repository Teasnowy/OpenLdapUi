# 组件库

- ## element-plus
```shell
npm install element-plus
```

- ## tailwindcss
> https://tailwindcss.com/docs/installation

```sh
# 安装vue依赖
npm install @headlessui/vue @heroicons/vue
# 安装本体
npm install -D tailwindcss postcss autoprefixer
# 初始化配置文件
npx tailwindcss init
# 安装daisyui拓展库
npm i -D daisyui@latest
```
